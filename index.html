Rotation Scoreboard はローテーション専用の無料スコアボードです。

上は得点、真ん中は履歴、下はボール入力と操作ボタンです。

画面は固定され、履歴エリアだけスクロールできます。

使い方

1. プレーヤー名とGoalを入力します。

2. 先攻プレーヤーを選びます。

3. Start Game で開始します。

4. 入ったボール番号をタップします。

5. 必要に応じて セーフティ / ファール / プレーヤー交代 / 戻す を使います。

6. 履歴は真ん中のエリアで確認します。

7. New で新しい試合を始めます。

履歴保存・勝率・平均得点などは、将来の有料版機能として検討中です。
